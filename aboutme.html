How This Website Was Built

A summary of the tools and techniques used in this assignment, as required under the Coding criteria.

ItemDetail
HTML Version HTML5 — semantic elements used throughout: <header>, <nav>, <main>, <section>, <article>, <aside>, <figure>, <figcaption>, <footer>
CSS Version External CSS3 only — one single stylesheet (css/style.css) linked across all 6 pages. No Bootstrap, no Tailwind, no inline styles for layout.
Text Editor Notepad / Notepad++ — no website builders, no WordPress, no Wix, no Dreamweaver used.
Fonts Google Fonts — Poppins (headings) and Open Sans (body text), with system-font fallback declared in CSS variables.
Colour Theme Deep Islamic green (#1A5C38) as primary colour; gold/amber (#C8A84B) as secondary colour — chosen to reflect the Islamic values of Nur Hidayah.
Layout Technique CSS Flexbox throughout — no CSS Grid, no Bootstrap grid. All responsive layout handled through flex-wrap and @media queries.
Link Types Used Internal links (navigation between all 6 pages), External links (Facebook, PEYATIM, Google Maps), Anchor links (jump to #map, #history, #gallery, #donation-instructions etc.), Image links (gallery images wrapped in <a> tags)
